## 🌟 About The Project

**Sohail Carpets** is a beautifully designed, fully responsive front-end landing page created for a premium rug and carpet business. It emphasizes a luxurious visual aesthetic, utilizing modern CSS layouts, smooth animations, and elegant typography to present a high-end catalog of woven art.

Whether you're looking for modern minimalism or vintage Oriental classics, this landing page is built to capture the essence of luxury decor.

## ✨ Features

- 💎 **Modern & Premium UI:** Clean layout with a sophisticated color palette designed to impress.
- 📱 **Fully Responsive:** Beautifully adapts to all devices (desktop, tablet, and mobile).
- 🎬 **Smooth Animations:** Subtle scroll and hover effects using CSS transitions and keyframes.
- 🧭 **Dynamic Navigation:** A sticky navbar that updates style on scroll and includes a mobile hamburger menu.
- 🧱 **Organized Structure:** Clean, maintainable, and semantic HTML5 and CSS3 code.
